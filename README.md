# webnovel-dl

Like YouTube-dl for webnovels.

## Installation

```sh
$ git clone https://github.com/jinyeow/webnovel-dl
$ cd webnovel-dl
$ crystal build src/webnovel-dl.cr --release
$ cp webnovel-dl ~/.bin
```

Or simple run `make`.

## Usage

See `webnovel-dl --help`.
See supported_sites.md for a list of providers.
